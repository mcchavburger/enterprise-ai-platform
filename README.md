# enterprise-ai-platform
A self-service enterprise AI platform built with Python, Docker, Kubernetes, Terraform, GitHub Actions and GitOps, progressing from local development to automated GPU-backed AI workloads.
# Enterprise AI Platform

> A production-style learning project that demonstrates modern Platform Engineering, DevOps and AI Infrastructure practices.

## Vision

The goal of this project is to design and build an enterprise-grade self-service AI platform from the ground up.

Rather than building isolated labs, this repository evolves over time into a complete platform that demonstrates how modern infrastructure is designed, automated, deployed and operated.

## Technology Roadmap

- ✅ Python
- ⏳ Docker
- ⏳ Kubernetes
- ⏳ Terraform
- ⏳ GitHub Actions
- ⏳ ArgoCD
- ⏳ Platform Engineering
- ⏳ AI Infrastructure

## Planned Features

- Python command-line interface (CLI)
- REST API
- Docker containerisation
- Kubernetes deployment
- Infrastructure as Code with Terraform
- GitHub Actions CI/CD
- GitOps with ArgoCD
- Monitoring with Prometheus & Grafana
- Local LLM deployment using Ollama
- Self-service AI environment provisioning

## Project Status

🚧 Under active development.

This repository is being built incrementally as part of a structured learning journey into Platform Engineering and Enterprise AI Infrastructure.
